# TopSpeed Automobile Management System

ASP.NET Core MVC application for managing automobile-related master data.

## Modules
- Brand Management
- Vehicle Type
- Posts
- Admin Panel (Role-based)

## Features
- CRUD operations
- Image upload & delete
- Role-based authorization
- Unit of Work & Repository pattern
- Logging and exception handling

## Technologies
- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Razor Views
- Dependency Injection

## Security
- Role-based authorization (Admin, MasterAdmin)

## Architecture
- Clean Architecture
- Repository Pattern
- Unit of Work
- Separation of Concerns

## Admin Features
- Brand image upload
- Secure delete with file cleanup
- Logging using ILogger
